# Healthcare AI Chatbot 🏥

An AI-powered healthcare chatbot built with the Anthropic Claude API. 
Simulates a friendly medical assistant that collects patient information, 
asks relevant follow-up questions, and provides possible conditions with 
detailed explanations.

## Features
- Multi-turn conversation with memory
- Collects patient name and generates unique Patient ID
- Asks follow-up questions before diagnosis
- Suggests possible conditions, tests, and home care tips
- Saves every consultation as a JSON file
- Always reminds users to consult a real doctor

## Tech Stack
- Python
- Anthropic Claude API
- Google Colab

## How to Run
1. Clone this repository
2. Install dependencies:
   pip install anthropic
3. Add your Anthropic API key
4. Run the notebook in Google Colab

## Important Disclaimer
This is an AI assistant and NOT a substitute for professional medical advice. 
Always consult a qualified healthcare provider for proper diagnosis.
